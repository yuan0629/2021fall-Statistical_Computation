    In either MCEM or MCNR, we use Metropolis algorithms to generate
\(N=10000\) values of \(u_1,\cdots,u_q\). The candidate distribution is
the marginal distribution of \(u_j\), that is, \(N(0,\sigma^2)\). Then
the acceptence rate is \[\begin{align}
    A_j(u, u^*)=\min\left\{1, e^{y_{+j}(u_j^*-u_j)}\prod_i \frac{1+e^{\beta x_{ij}+u_j}}{1+e^{\beta x_{ij}+u_j^*}}\right\} 
\end{align}\] In the simulation study, we set the first \(N_1=500\)
brun-in and use only the last \(N-N_1\) samples.

    In MCNR, we use Newton-Raphson iteration to get \(\beta\)
\[\begin{align}
    \beta^{(m+1)}=\beta^{(m)}+E[X'W(\beta^{(m)},U)X|y]^{-1}X'(y-E[\mu(X'W(\beta^{(m)},U)|y])
\end{align}\] where \(\mu_i(\beta,u)=1/(1+\exp\{-\beta x_{ij}-u_j\})\),
\(W(\beta,u)=diag\{\mu_i(\beta,u)(1-\mu_i(\beta,u))\}\).
